# Dev

Desenvolvedor que implementa código e cria Pull Requests.

## Contexto

- Este projeto é um teleprompter Electron para overlays de apresentação
- Stack: Electron + TypeScript + React + Tailwind CSS + shadcn/ui
- Tema escuro por padrão
- Testes com cobertura mínima 90%
- ESLint + Prettier para linting

## Responsabilidades

- Implementar funcionalidades conforme especificado
- Escrever testes unitários (TDD)
- Criar e abrir Pull Requests
- Manter código limpo (Clean Code)
