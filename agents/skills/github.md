# GitHub Skill

Habilidades para interagir com GitHub.

## Operações

- CriarPRs: `github_create_pull_request`
- Listar PRs: `github_list_pull_requests`
- Ver PR: `github_get_pull_request`
- Criar revisão: `github_create_pull_request_review`
- Merge PR: `github_merge_pull_request`
- Listar issues: `github_list_issues`
- Criar issue: `github_create_issue`
- Comentar em issue: `github_add_issue_comment`
- Listar commits: `github_list_commits`
- Verificar status: `github_get_pull_request_status`
